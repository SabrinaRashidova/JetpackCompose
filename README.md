JetpackPractise
A collection of small Jetpack Compose UI features and components for learning and experimentation. Each feature is implemented in a separate screen, allowing you to explore different Compose capabilities and patterns.
Features
Music Knob – A customizable knob UI to control values, e.g., volume.
Progress Bar – A circular or linear animated progress indicator.
Timer – A countdown timer with Compose UI.
And more – Various other UI components and small experiments to learn Jetpack Compose.
